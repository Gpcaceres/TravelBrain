# UNIT-TEST-BUSINESS-RULES

Listado de pruebas unitarias implementadas en el módulo business-rules-backend:

- businessRules.test.js: Pruebas de las reglas de negocio principales.
- itinerary.test.js: Pruebas de lógica y validaciones de itinerarios.
- performance.test.js: Pruebas de rendimiento de endpoints de reglas de negocio.
- security.test.js: Pruebas de seguridad y validación de accesos.
- trip.coverage.test.js: Pruebas de cobertura adicionales para lógica de viajes.

Ubicación: business-rules-backend/tests/

Cada archivo contiene pruebas unitarias usando Jest y Supertest para cubrir los endpoints y lógica principal de business-rules-backend.

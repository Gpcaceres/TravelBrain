# UNIT-TEST-BACKEND

Listado de pruebas unitarias implementadas en el backend:

- auth.test.js: Pruebas de autenticación y autorización.
- performance.test.js: Pruebas de rendimiento de endpoints principales.
- security.test.js: Pruebas de seguridad y validación de accesos.
- trips.test.js: Pruebas CRUD de viajes (Trip).
- trips.coverage.test.js: Pruebas de cobertura adicionales para Trip.
- users.test.js: Pruebas CRUD de usuarios (User).
- destinations.test.js: Pruebas CRUD de destinos (Destination).
- weather.test.js: Pruebas de obtención de clima (Weather).
- itineraries.test.js: Pruebas CRUD de itinerarios (Itinerary).
- favoriteRoutes.test.js: Pruebas CRUD de rutas favoritas (FavoriteRoute).

Ubicación: backend-project/tests/

Cada archivo contiene pruebas unitarias usando Jest y Supertest para cubrir los endpoints y lógica principal del backend.

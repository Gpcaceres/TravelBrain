# QUICKSTART - Desarrollo Local

## Ejecutar (Windows):
```cmd
check-local.bat
start-local.bat
```

## Ver logs:
```cmd
logs-local.bat facial-recognition
```

## URLs:
- Frontend: http://localhost
- API: http://localhost/api
- Face API: http://localhost/api/face
- Face Docs: http://localhost:8000/docs

## Reconstruir solo facial recognition:
```cmd
rebuild-face.bat
```

## Test:
```cmd
test-face-service.bat
```

## Detener:
```cmd
stop-local.bat
```
